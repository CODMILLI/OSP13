# <i>"Moodsic Tracker"</i>
2020-02 Open Software Platform class, Project of Team13<br>
<b>2020.10~2020.12</b><br>
<i>music streaming + personal diary Website</i>

## Project Description
Some people say that when they listen to a song that they used to listen to a lot again, the memories of that time revived vividly. <br>
We combined music streaming application and diary application to make recording a daily log easy and special.

We got an idea from a <i>mood tracker</i> that records a day with mood, <br>
and named it in the sense of recording a day with music that can express feelings, memory, etc... of the day.

## Website Composition
<b>Login Page</b>
- 로그인/회원가입

<b>Music Player</b>
- 재생/일시정지
- 이전곡/다음곡
- 셔플/반복 on/off
- 소리 조절
- 재생바 조절

<b>Main Page</b>
- 오늘 내가 기록한 일기를 볼 수 있음 (그에 따라 테마도 변경)
- 인기 태그 노출
- 검색창에서 음악 검색 가능
- 버튼을 눌러 바로 오늘의 일기 작성 가능

<b>Diary List Page</b>
- 월별로 달력형/목록형 다이어리 볼 수 있음
- 원하는 날짜를 클릭하면 해당 날짜에 대한 일기 기록 가능

<b>Diary Page</b>
- 일기 열람/수정/삭제
- 처음 작성시 플레이리스트에서 노래 선택
- 태그 및 내용 작성

<b>Diary Analysis Page</b>
- 작성한 일기의 태그, 노래 바탕으로 분석 결과 제공

<b>Music Recommandation Page</b>
